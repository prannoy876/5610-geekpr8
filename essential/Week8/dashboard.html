About the Experiment

      In this experiment I went about designing my dashboard page that shows up
      when the user logs in.This is not the user's profile page but the home page which
      provides the user with easy access to other features of the web site.A lot of
      TIME and PAINS have been taken to develop this page but it is by no means complete.I 
      will constantly be making it better in future experimental itertions.Have a peak
      at the dashboard.css files for more information.I am also using the MARVEL Developer API
      to create the API.

angular.module('MarvelApp').controller('DashboardController', ['$scope', '$http', '$location', '$rootScope', function ($scope, $http, $location, $rootScope) {

    $scope.name = $rootScope.currentUser.fname;
    $scope.logout = function () {

        $http.post("/logout")
    .success(function (response) {
        $location.url("/login");

    });
    }
} ]);
