About the Experiment

     In this experiment,I have created  the user's dedicated profile page.
     It basically keeps track of all the user's interactions with the web page.
     A user can edit the list of characters that he likes and can also
     see the list of comics that he recently reviewed.For a future experiment i will 
     let a user view another user's profile page.
