About the Experiment

       In this experiment,I demonstrate the use of Angular Directives.With the help of
       angular directives you can have greater control of your angular app and over the 
       DOM.In this experiment I am making use of Carousel which displays the images
       of all comics belonging to a particular character.But I cannot display the carousel 
       until all the images have been recieved from the API(Asynchronous).I could set a timeout
       before the carousel is loaded but still there is no guarantee.By using a user-defined directive
       I listen to "LAST" angular-repeat which loads the carousel list which fires the directive which then
       in turn,fires the jquery function which generates the carousel that refers to the list(That is
       already loaded previously by ng-repeat).

Javascript(Angular)

       angular.module('MarvelApp').directive('onLastRepeat', function () {
    return function (scope, element, attrs) {
        if (scope.$last) setTimeout(function () {
            scope.$emit('onRepeatLast', element, attrs);
        }, 10000);
    };
})


 $scope.$on('onRepeatLast', function (scope, element, attrs) {
        console.log('Executing');
        setTimeout(function () {
            $("#owl-demo").owlCarousel({
                autoPlay: 3000,
                stopOnHover: true,
                responsive: {
                    0: {
                        items: 1
                    },
                    600: {
                        items: 3
                    },
                    1000: {
                        items: 3
                    }
                },
                navigation: true,
                paginationSpeed: 1000,
                goToFirstSpeed: 2000,
                autoHeight: true,
                transitionStyle: "fade"
            });
        }, 3000);

        $timeout(function () { $scope.loaded = true; }, 2000);

    });
