CSS Transitions

Move your mouse over here

     Usually we use Javascipt or
     JQuery to perform custom animations but CSS is so powerful that we can
     now perform complex animations using just CSS and do the same things
     that we used to do using javascipt or Jquery.

CSS

 .animate {
    width: 100px;
    height: 100px;
    background: #9FFEF5;
    -webkit-transition: width 3s, height 4s; /* For Safari 3.1 to 6.0 */
    transition: width 3s, height 4s;
    margin:10px;
}

.animate:hover {
    width: 300px;
    height: 300px;
}
