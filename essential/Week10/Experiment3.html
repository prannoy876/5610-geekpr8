About the Experiment

     In this experiment I will teach you how to add zoom
     in over images upon the the mouse hover option.We can enchance user
     experience by animating the images when the user moves the mouse
     over them.In this case I have used the class "poster" along with the img tag.

CSS


 .poster
{
    transition: all .2s ease-in-out;
    display:block;
   display: inline-block;
    vertical-align: middle;
}
.poster:hover {
    position:relative;
    z-index:999;
    transform: scale(1.50);
    cursor:pointer;
}
