About the Experiment

      This is a really fun experiment!Previously I showed you how to
      bring up a blocking dialogue using Bootstrap.This time I will teach you how
      to create your own custom alert/error dialogue which blocks the user from
      accesing the other DOM elemnts unless he/she dismisses the pop up dialogue.

HTML5


 <div id="overlay-for-contacts" >

            <div class="alert alert-info" id="overlay-for-contacts-inside">

            <p class="info"><h4>This is the Demo</h4></p>

            <button type="button" class="btn closebutton btn-default"  value="Close">Close</button>

            </div>

    </div>

Javascript(JQuery)

    $(document).ready(function () {
    $("#overlay-for-contacts").hide();



    $(".closebutton").click(function (e) {
        e.preventDefault();
        $("#overlay-for-contacts").fadeOut("slow");
    });

    $(".demo").click(function (e) {
        e.preventDefault();
        $("#overlay-for-contacts").fadeIn("slow");
        $(window).scrollTop($("#overlay-for-contacts").offset().top);
    });


});
