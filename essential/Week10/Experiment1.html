About the Experiment

      In this experiment, I show you how a session can be terminated when
      a user logs out.When the logout button in clicked on the tab,what happens
      is that req.logout() line is executed which deactivates the session-cookie which
      is associated with that particular user thus ending the session and sending a
      200 Ok response.When I recieve the response I redirect the user to the dashboard
      but only this time the user is not loggedin and some features that were 
      previously made available to the user are now not available.

      Steps to execute the experiment--------------

      1.Log in and look at the Dashboard which recognizes that the user is logged in
      2.Logout and see how the options on the dashbaord change
      3.Try adding items to cart/add favorite characters-----it will direct you to the login page now!.
