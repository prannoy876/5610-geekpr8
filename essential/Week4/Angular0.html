My Name is{{person.name}}.I live in {{person.city}}.My email is {{person.email}}

About the Experiment


         In this experiemnt,I briefly deomstrate how angular-js can be used to submit a form
         and capture the values in the fields.For this we make use of mg-model.In the form i have assigned
         ng-model="user.name" ,"user.city" and so on.And on submit I am sending the whole user object to
         the control which assigns it the scope.person.The controllers $scope is the bridge between
         the view and the controller.

HTML

 <div class="main-container" ng-controller = "myController3">

<form>

  <div class="form-group">

  <div class="form-group">

    <label for="exampleInputPassword1">Name</label>

    <input type="text" ng-model="user.name" class="form-control" id="exampleInputPassword1" placeholder="Password">

  </div>

  <div class="form-group">

    <label for="exampleInputPassword1">City</label>

    <input type="text" ng-model="user.city" class="form-control" id="Text1" placeholder="Password">

  </div>

    <label for="exampleInputEmail1">Email address</label>

    <input type="email" ng-model="user.email" class="form-control" id="exampleInputEmail1" placeholder="Enter email">

  </div>

  

  <button type="submit" ng-click="submit(user)"class="btn btn-default">Submit</button>

   <p></p><h4> My Name is{{person.name}}.I live in {{person.city}}.My email is {{person.email}}</h4>

</div>

Javascript

var app = angular.module("App", []);


app.controller("myController3", function ($scope, $http) {
   
    $scope.person = null;

    $scope.submit = function (user) {

        $scope.person = user;

    }



});
