About the Experiment

        This experiment is actually the sign up page of my project.I am using
      angular the angular js routing to load this page on '/signup' url.

        Once all the fields have been validated the sign up button is
      activated and all the data is stored in mongo-db table back end.


       To make sure if you sign up worked,Move on to the next experiemnt which will
       be the Login page.I will iteratively be improving User Experience on these
       pages such as letting the user know if the sign-up worked or not.
